# 명령어

```
:%s/aaa/bbb/g
```

# 뜻

aaa를 모두 bbb로 바꿔라

# 응용

```
:%s!aaa!bbb!g
```

기본 구분자인 슬래시(/) 대신 다른 구분자를 사용할 수 있다.

변경 하려는 문구에 슬래시(/)가 이미 들어가 있을 경우 유용하다.

vi 편집기 밖에서 파일에 있는 문자열을 수정할때 쓰는 sed 명령어와 구성이 매우 유사하다.
