title: Git Branch 관련 명령어
publishedAt: 2025-07-28
categories: ZA,Git
---

로컬의 모든 브랜치 목록을 볼 수 있다.

```
git branch
```

원격의 모든 브랜치 목록을 볼 수 있다.

```
git branch -r
```

로컬과 원격의 모든 브랜치 목록을 볼 수 있다.

```
git branch -a
```

원격 브랜치를 로컬로 최신화한다.

```
git remote update
```

새로운 브랜치를 만듬과 동시에 이동한다.

```
git checkout -b 새로운브랜치명
```

브랜치를 가져옴과 동시에 이동한다.

```
git checkout -t origin/브랜치명
```
